🚖 Taxi Booking Application
📌 Project Description

The Taxi Booking Application is a Java-based console application that simulates a basic taxi booking system. The system allows customers to book taxis, and taxis are assigned based on availability and nearest location. The goal of this project is to demonstrate core Java concepts such as object-oriented programming, collections, and basic logic building.

This project helps in understanding how real-world taxi services manage bookings, allocate drivers, and calculate fares.

🎯 Features

Customer can book a taxi.

Automatic taxi allocation based on availability.

Displays booking details.

Calculates fare based on travel distance.

Shows taxi booking history.

🛠️ Technologies Used

Java

Object-Oriented Programming (OOP)


📂 Project Structure

TaxiBookingApplication – Main class to run the application.

Taxi – Manages taxi details and availability.

Customer – Stores customer booking information.

Booking – Handles booking details and fare calculation.

🚀 How to Run

Clone the repository.

Open the project in Eclipse or any Java IDE.

Compile and run the TaxiBookingApplication class.

Follow the console instructions to book a taxi.
